
# Production Deployment Checklist

## Infrastructure
- [ ] AI Search replicas scaled for peak load
- [ ] OpenAI provisioned throughput allocated
- [ ] Redis caching operational
- [ ] CDN for static assets

## Monitoring
- [ ] Latency dashboards (P50/P95/P99)
- [ ] Cost alerts (< target budget)
- [ ] Error rate < 1%
- [ ] Application Insights integrated

## Quality Gates
- [ ] Test suite passes (>85%)
- [ ] Canary deployment successful (10% traffic)
- [ ] Security review signed off
- [ ] Content owners notified

## Rollout
- [ ] User training materials ready
- [ ] Support escalation path defined
- [ ] Rollback plan documented
